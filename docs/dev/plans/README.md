# Plans

Plans for non-trivial work. Each plan must be referenced from [`../tasklist.md`](../tasklist.md).

A plan should cover:

- Goal
- Scope
- Non-goals
- Approach
- Affected modules
- Tests
- Docs impact
- i18n impact
- Accessibility impact
- Risks
- Open questions

See [`AGENTS.md`](../../../AGENTS.md) for the rules.
